Detailed Description

Paths to map properties in a Tiled json file.

All paths here use JSON Paths syntax.

Some paths use %d conversion specifiers as placeholders for the index of a discrete value being iterated over.

Macro Definition Documentation

◆ TILEMAP_JPATH_SCRIPT

#define TILEMAP_JPATH_SCRIPT   "/properties/[name=script_%d]/value"

JSON path to script definition.

Scripts are stored in map properties as "script_n", where n represents the index of the script as referred to by MOBILE::script_id.

Scripts are stored as sequences of a single character (shorthand for a statement) followed by a number (an argument). They must be parsed by script_parse_str.
